# Python Lists

## Python Lists

### Python List

- `[a, b, c]`

## Subsetting Lists

### List slicing

`[ start : end ]`

- 'start' inclusive; 'end' exclusive

## Manipulating Lists

### Changing list elements

`fam[7] = 1.76`

`fam[0:2] = ['lisa', 1.84]`

### Adding and removing elements

`fam + ['me', 1.54]`

- Concatenates the two lists

`del(fam[2])`

- Deletes the 3rd element from the list
